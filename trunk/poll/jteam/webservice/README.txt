1) download axis http://apache.infocom.ua/ws/axis2/1_4_1/axis2-1.4.1-bin.zip
2) unzip downloaded file
3) enter /repository/ in axis directory, create a folder named "servicejars"
4) copy axis2.xml to /conf/axis2.xml
5) start the server via /bin/axis2server.bat
6)goto http://localhost:8080/ to see the list of services available
