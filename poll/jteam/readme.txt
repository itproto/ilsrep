Use scripts in /release directory to launch program(s):
client.{sh, bat}	- console client
editor.{sh, bat}	- console editor
pollApplicationGUI.{sh, bat}	- GUI client
editorGUI.{sh, bat}	- GUI editor(will be merged with GUI client)
server.{sh, bat}	- server
configurationEditor.{sh, bat}	- configuration editor for server
